### 提交信息规范

使用webstorm请安装 https://plugins.jetbrains.com/plugin/13389-conventional-commit 插件以便获得更好的智能提示

在项目`conventionalcommit.json`中配置

type(scope): body

type: 提交类型
scope：影响到的模块
body: 提交信息简写
